# Legs
Biped robot using Jetson Nano

## Hardware
- Jetson Nano
- 10x Dynamixel AX-12A actuators
- Intel RealSense D435 depth camera
- TBD IMU
- TBD Battery/Power Distribution

## Software
- Ubuntu 20.04
- ROS2 Galactic
